Customer Management API Documentation

This API allows for managing customer records, including creating, retrieving, updating, and deleting customer information.

API Endpoints

Retrieve a Customer (GET)

Endpoint: /customers/{id}

Method: GET

Description: Retrieves details of a specific customer by ID.

Response: JSON object containing customer details.

Retrieve All Customers (GET)

Endpoint: /customers

Method: GET

Description: Retrieves a list of all customers.

Response: JSON array of customer objects.

Add a New Customer (POST)

Endpoint: /customers

Method: POST

Description: Adds a new customer record.

Request Body: JSON object containing the new customer's details.

Response: JSON object of the added customer.

Update an Existing Customer (PUT)

Endpoint: /customers/{id}

Method: PUT

Description: Updates an existing customer's details.

Request Body: JSON object containing the customer's updated details.

Response: JSON object of the updated customer.

Delete a Customer (DELETE)

Endpoint: /customers/{id}

Method: DELETE

Description: Deletes a specific customer by ID.

Response: JSON object of the remaining customers after deletion.

Error Handling

API uses standard HTTP status codes to indicate the success or failure of requests.

  • 200 OK: The request was successful.
  • 400 Bad Request: The request was malformed or invalid.
  • 404 Not Found: The specified resource was not found.
  • 500 Internal Server Error: An error occurred on the server side.
